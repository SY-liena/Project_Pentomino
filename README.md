# "전공 로드맵 시각화 시스템"  by 소연

안녕하세요 :) `Flask` 기반 백엔드와 프론트엔드 시각화 UI를 결합한 `전공 로드맵 시각화 시스템` 프로젝트 입니다!
사용자는 부산대학교 전공 커리큘럼을 학년별 로드맵 형태로 확인할 수 있고, AI 챗봇을 통해 전공 상담을 받을 수 있습니다~!

---

## 주요 구성 요소

- `app.py`
  - Flask 서버 엔드포인트를 제공합니다.
  - `google.generativeai` 라이브러리를 사용해 Gemini AI와 연동합니다.
  - `/ask-ai` POST 요청을 받아 사용자 질문을 AI로 전달하고 답변을 반환합니다.
  - `DEBUG` 모드를 통해 개발 중 에러 상세 정보를 확인할 수 있습니다.

- `index.html`
  - 웹 페이지의 기본 레이아웃과 AI 챗봇 UI, 로드맵 선택 기능을 포함합니다.
  - 채팅창 토글, 학과 선택 드롭다운, 로드맵 표시 영역을 구성합니다.

- `script.js`
  - `data.json`에서 학과 데이터를 불러와 드롭다운 메뉴와 학년별 로드맵을 동적으로 렌더링합니다.
  - AI 챗봇 사용자 인터랙션, 서버 호출, 에러 핸들링을 구현합니다.
  - `AI_ENDPOINT`를 통해 배포된 Flask 서버로 질문을 전달합니다.

- `style.css`
  - 페이지 전반의 디자인과 레이아웃을 정의합니다.
  - AI 챗봇 창, 네비게이션 버튼, 로드맵 카드 스타일을 적용합니다.

- `data.json`
  - 학과 및 커리큘럼 데이터를 저장합니다.
  - `script.js`에서 로드맵 생성 데이터로 사용합니다.

- `requirements.txt`
  - 실행에 필요한 Python 패키지를 정의합니다.

- `Procfile`
  - Render 또는 유사 PaaS 환경에 배포할 때 사용합니다.

## 기능 요약

- 학과 선택 시 1~4학년 전공 로드맵을 시각적으로 표시
- `전공필수`, `전공기초`, `전공선택` 과목별 구분 표시
- AI 챗봇 기능을 통해 전공 상담 질문 및 답변 제공
- AI 서버 통신 실패 시 UI에서 에러 메시지 출력

## 적용된 개선 사항 및 문제 해결 과정

1. **조건 검증과 안정성 강화**
   - `app.py`에서 `GOOGLE_API_KEY`가 없는 경우 `ValueError`를 발생시켜 배포 환경에서 키 누락 문제를 빠르게 파악하도록 개선했습니다.
   - JSON 요청이 올바른지 확인하고, 비어 있는 질문이 들어오면 사용자에게 명확한 피드백을 제공합니다.

2. **Gemini 모델 선택 로직 추가**
   - 사용 가능한 Gemini 모델 목록을 확인하고 `generateContent`를 지원하는 모델 중 우선순위에 따라 모델을 선택하도록 구현했습니다.
   - 기본 `gemini-1.5-flash` 모델이 없을 경우에도 다른 지원 모델을 자동 선택하여 서비스 가용성을 높였습니다.

3. **프론트엔드 오류 처리 개선**
   - `script.js`에서 HTTP 응답 상태를 검사하고, 서버 오류 메시지 또는 JSON 파싱 오류에 대비했습니다.
   - 로딩 상태 표시를 추가하여 사용자가 AI 응답 대기 중임을 명확히 알 수 있도록 했습니다.

4. **UI/UX 개선**
   - `style.css`에 AI 챗봇 버튼 스타일과 채팅창 레이아웃을 추가하여 전체 UI가 더욱 통일되고 현대적인 느낌이 들도록 개선했습니다.
   - 네비게이션 메뉴의 버튼과 간격을 조정해 사용자 인터페이스를 한층 다듬었습니다.

5. **배포 준비**
   - Flask 앱이 `PORT` 환경 변수를 사용하도록 설정하여 Render와 같은 플랫폼에 쉽게 배포할 수 있도록 했습니다.
   - `requirements.txt`와 `Procfile`을 통해 배포 환경 구성이 명확히 드러납니다.

## 실행 방법

1. Python 환경 준비

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

2. 환경 변수 설정

- `.env` 파일 또는 배포 환경 변수에 `GOOGLE_API_KEY`를 추가합니다.
- 로컬에서 디버그 정보를 보고 싶으면 `DEBUG=true`를 설정합니다.

3. 서버 실행

```bash
python app.py
```

4. 브라우저에서 확인

- `http://127.0.0.1:5000` 에 접속하여 전공 로드맵 및 AI 상담 기능을 확인합니다.
